# Architecture Decision Records

## ADR-001: Documentation First
Status: Accepted

Guardian must be designed from documented contracts before implementation.

## ADR-002: Security Kernel Priority
Status: Accepted

AntiNuke, authority, containment, recovery, and forensics always outrank optional modules.

## ADR-003: Evidence-Based Certification
Status: Accepted

No feature is certified without tests, logs, drills, and repeatable validation.
