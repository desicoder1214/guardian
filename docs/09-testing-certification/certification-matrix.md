# Certification Matrix

## Purpose
Define what must pass before Guardian features are certified.

| Area | Requirement | Evidence |
|---|---|---|
| AntiNuke | Detect destructive action | Logs + drill |
| Authority | Correct trust decision | Tests + audit |
| Bot Authorization | Remove unauthorized bot | Live drill |
| Webhook Security | Freeze unauthorized webhook | Drill logs |
| Lockdown | Real permission denial | Reality audit |
| Recovery | Restore structure | Recovery report |
| Logging | Forensic trace exists | Log review |

## Certification Rule
A feature is not certified until evidence proves it works.
