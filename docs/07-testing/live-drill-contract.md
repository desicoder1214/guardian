# Guardian Live Drill Contract

## 1. Purpose

This contract defines Guardian's live operational validation process.

Live drills provide production-like validation of Guardian's security capabilities under controlled conditions before release certification.

---

## 2. Global Rules

- Live drills must be executed only in dedicated test guilds.
- No production community may be used without explicit authorization.
- Every drill must produce complete forensic evidence.
- Every failure must become a regression test.
- Critical failures block release certification.
