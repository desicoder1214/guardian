# Active Nuke Response Runbook

## Purpose

Defines response steps during active destructive guild attack.

## Steps

1. Confirm destructive activity.
2. Activate containment.
3. Remove malicious bot or punish actor.
4. Freeze webhooks.
5. Activate lockdown if needed.
6. Start recovery only after containment.
7. Review logs and recovery results.

## Rule

Containment comes before recovery.
