# Live Drill Execution Runbook

## Purpose

Defines how to execute controlled Guardian certification drills.

## Steps

1. Use isolated test guild.
2. Confirm snapshots.
3. Confirm test accounts.
4. Confirm logging routes.
5. Execute drill steps.
6. Capture evidence.
7. Record pass/fail verdict.
8. File follow-up work.

## Rule

No destructive drill runs in a production community guild.
