# Webhook Compromise Response Runbook

## Purpose

Defines response steps for webhook abuse or persistence.

## Steps

1. Identify webhook.
2. Identify channel.
3. Attribute creator if possible.
4. Delete or freeze unauthorized webhook.
5. Lock affected channel if needed.
6. Review related bot or admin activity.
7. Preserve evidence.

## Success Criteria

Webhook removed or frozen and forensic evidence recorded.
