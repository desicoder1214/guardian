# Anti-Drift Feature Ledger

This ledger prevents Guardian from drifting away from its security mission.

- AntiNuke
- AntiSpam
- Moderation
- FakePerms
- Unauthorized Bot Protection
- Webhook Security
- Invite Protection
- Lockdown
- Recovery
- Logging
- Certification

## Optional Future Features
- AI moderation
- Leveling
- Giveaways
- Tickets
- Analytics
- Dashboard

## Rule
Optional features must never slow or weaken the security kernel.
