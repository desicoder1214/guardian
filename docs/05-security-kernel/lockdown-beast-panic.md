# Lockdown, Beast Mode, and Panic

## Purpose
Stop active raids or nukes by restricting damage paths.

## Modes
- Single Channel Lock
- Server Lock
- Panic Mode
- Emergency Overlay
- Recovery Lock

## Requirements
- Lock must enforce real Discord permissions.
- Unlock must restore previous state.
- Public UI must show truthful status.
- Normal users must be unable to bypass lock.

## Rule
Lockdown must prioritize containment speed and restore correctness.
