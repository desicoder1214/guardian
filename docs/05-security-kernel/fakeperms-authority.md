# FakePerms Authority

## Purpose
Guardian must evaluate trusted authority separately from raw Discord permissions.

## Principles
- Discord Administrator does not automatically mean Guardian trust.
- Owner and approved trusted users are explicit.
- Dangerous actions require Guardian-level authority.
- Trust must be inspectable and revocable.

## Capabilities
- Security Admin
- Moderator
- Recovery Operator
- Bot Authorizer
- Lockdown Operator

## Rule
FakePerms decisions must be logged and testable.
